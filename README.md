# zotero-mcp

Read+write MCP server for Zotero. First MCP server with full write support for managing your Zotero library from AI assistants.

## Features

- Search, read, and export items (JSON, BibTeX)
- Create items from DOI, BibTeX, or manual entry
- Manage collections (create, add/remove items)
- Find and merge duplicate items
- Works with local Zotero desktop API and Zotero Web API

## Installation

```bash
uv tool install zotero-mcp-rw
```

## Usage with Claude Code

```bash
claude mcp add zotero -- zotero-mcp
```
